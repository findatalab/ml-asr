# ml-asr
Разработка моделей машинного обучения детекции и фильтрации аудиосигналов для улучшения качества распознавания речи

Пока разработано веб-приложение для подавления шумов в речевых сигналах в режиме реального времени. Система использует архитектуру нейронной сети Band-Split RNN (BSRNN) для фильтрации аудиозаписей от различных типов помех.
Основные функции

    Запись аудио: Захват голосового потока через интерфейс MediaRecorder API непосредственно в браузере.

    Симуляция зашумления: Программное наложение трех типов помех:

        Babble (фоновые разговоры);

        RIR (реверберация и комнатный отклик);

        White Noise (белый шум).

    Denoising: Обработка зашумленного сигнала с помощью нейросетевой модели BSRNN.

    Автоматическая очистка: Система удаления временных файлов для оптимизации дискового пространства сервера.

Технологический стек

    Backend: Python 3.9+, Flask, PyTorch.

    Обработка аудио: Torchaudio (STFT/ISTFT), Soundfile.

    Frontend: JavaScript (Vanilla JS), HTML5 Audio API.

    Модель: Архитектура BSRNN.

Инструкция по развертыванию
1. Подготовка окружения

Рекомендуется использовать изолированное виртуальное окружение:

python -m venv venv

venv\Scripts\activate 

2. Установка зависимостей

pip install torch torchaudio flask soundfile

3. Размещение весов модели

Файл с весами bsrnn_weights_final.pth должен быть расположен в директории models/. При отсутствии файла модель будет инициализирована случайными весами.
4. Запуск приложения

python app.py

Доступ к интерфейсу осуществляется по адресу: http://127.0.0.1:5000
